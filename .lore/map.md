# Lore architecture map

_Generated by Lore. Use this to find which files/areas a change concerns before reading code._

- Frontend — macOS menu bar app (SoniqueBarApp, SettingsView) for voice interaction, wake-word capture, and UI
- Backend — Service layer (SoniqueBrain, IntentRouter, CommandServer) plus a Python Kokoro TTS HTTP service for on-device speech synthesis
- Integrations — LLM, voice/TTS providers, and external tool connectors driving assistant capabilities
- Infrastructure — Build, deploy, and sidecar packaging scripts plus xcodeproj generation and CI helpers
- CLI — KokoroCLI command-line entry point for running the TTS engine standalone
