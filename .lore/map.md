# Lore architecture map

_Generated by Lore. Use this to find which files/areas a change concerns before reading code._

- Frontend — Menu-bar app UI (SoniqueBarApp, SettingsView) plus screen capture/awareness and notification surfaces
- Backend — Swift service layer (brain, intent routing, orchestration, command server) and a local Python Kokoro TTS sidecar exposing an HTTP API
- Integrations — Connectors to external dev tools and pluggable LLM/voice providers consumed by the assistant
- Infrastructure — Build, sidecar packaging, deploy and watcher scripts; on-device ML runtime (mlx-swift, kokoro-swift) and model conversion
- CLI — KokoroCLI command-line entry point for running TTS synthesis directly
